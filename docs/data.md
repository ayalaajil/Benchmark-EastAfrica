# Data & Downloads

Everything on this site is generated by the pipeline in this repository; none
of it is hand-edited. This page indexes what's available to download directly
and what you can regenerate yourself.

## Figures on this site

Every figure shown in [Key Findings](key-findings.md) and
[Results](results/index.md) is a static PNG committed to the repository under
[`docs/assets/figures/`](https://github.com/ayalaajil/Benchmark-EastAfrica/tree/main/docs/assets/figures)
— right-click any figure on this site and "save image" to get a full-resolution
copy, or browse the folder directly on GitHub.

Every figure is written twice by the pipeline — vector PDF (for papers) and
300-dpi PNG (shown here).

| File | Shows |
|---|---|
| `timeseries.png` | Domain-mean daily rainfall by lead day, all models vs all references |
| `temporal_bias_mae.png` | Rolling bias & MAE vs valid date, by lead day |
| `ensemble_crps_spread_ssr.png` | CRPS, spread and spread–skill over the season — GenCast & NeuralGCM, one curve per lead day |
| `spatial_maps_chirps_ld1.png` / `_ld7.png` | Per-cell bias & RMSE maps vs CHIRPS |
| `acc_lead_curves.png` | Anomaly correlation vs lead day (CHIRPS & TAMSAT panels) |
| `ssr_lead_curves_chirps.png` | Ensemble spread–skill ratio vs lead |
| `ssr_zonal_chirps.png` | Ensemble spread–skill ratio by latitude |
| `rank_histograms_gencast.png` / `_neuralgcm.png` | Talagrand rank histograms (tie-aware ranks), by reference and lead |
| `reliability_gencast.png` / `_neuralgcm.png` | Reliability diagrams at local percentile thresholds |
| `crpss_maps_chirps.png` | CRPS skill score vs climatology, per cell, models × leads |

## CSV tables

The verification pipeline (`run_verification.py`) writes the full numeric
results — the tables summarized on the
[Results](results/index.md) pages — to `mam2024_analysis_outputs/`. These CSVs
are not committed to the repository (they're regenerated from predictions,
which are themselves large Zarr stores), but every one is reproducible with a
single command. Key files:

| File | Contents |
|---|---|
| `crpss_vs_climatology_by_model_obs_lead.csv` | CRPSS per model × reference × lead |
| `event_scores_by_threshold.csv` | POD/FAR/CSI/frequency bias per model × threshold × lead |
| `probabilistic_scores.csv` | CRPS, spread and spread–skill per ensemble model × reference × lead |
| `brier_scores.csv` | Brier scores per ensemble model × threshold |
| `reliability_tables.csv` | Reliability-diagram bins and ECE per ensemble model × threshold/reference |
| `acc_by_model_truth_lead.csv` | Pooled anomaly correlation per model × truth × lead |
| `ssr_by_model_truth_lead.csv` | Fortin-corrected spread, RMSE and SSR per ensemble model × truth × lead |

## Regenerating everything

All inputs are public — model checkpoints and ERA5 initial conditions are read
from anonymous Google Cloud Storage, and CHIRPS/ERA5/TAMSAT are downloaded and
cached on first use. Full commands to regenerate predictions, tables and
figures from scratch are on the [Reproducibility](reproducibility.md) page.
